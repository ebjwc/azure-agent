# azure-agent

This repository describes an azure devops build agent dockerfile with following technologies installed:
* Ubuntu 16.04
* Open JDK 8
* Ant
* Gradle 4.6
* Maven 3.6.0
* ActiveMQ 5.15.6
* Postgres
* Graphviz
* Docker 17.12.0-ce

To use this docker image type:
`docker run jhageste/azure-agent`
